  {
    data: [
      ["Anna", 16, "Female", "2004-12-01"],
      ["Kevin", 22, "Male", "1998-01-21"]
    ],
    column: [
      { name: "name", note: '' },
      { name: "age", note: '' },
      { name: "gender", note: '' },
      { name: "birthday", note: '' }
    ]
  }

data 是数据，column 是对数据的说明。现写一个函数 parseData，将上面的对象转化为期望的数组以供前端展示所使用：


    [
      { name: "Anna", age: 16, gender: "Female", birthday: "2004-12-01" },
      { name: "Kevin", age: 22, gender: "Male", birthday: "1998-01-21" },
    ]

Practice2

有一个存放字符串数据的数组：


    ['item1', 'item2', 'item3', 'item4', 'item5']

有另一个数组存放对象如下：


    [
      { content: 'section1', index: 0 },
      { content: 'section2', index: 2 }
    ]

这个数组表示它每个对象会往原来的数组的 index 坐标前插入 content 数据（index 不会重复）：


         0      1      2      3      4
       item1  itme2  item3  item4  item5
      ^             ^ 
      |             |
   section1     section2  

  最后结果是：['section1', 'item1', 'item2', 'section2', 'item3', 'item4', 'item5']

请你完成 inject 函数，以达到上述的功能：


    inject(
      ['item1', 'item2', 'item3', 'item4', 'item5'],
      [
        { content: 'section1', index: 0 },
        { content: 'section2', index: 2 }
      ]
    ) // => ['section1', 'item1', 'item2', 'section2', 'item3', 'item4', 'item5']

练习文件 /src/practice2.js

请尽可能使用课程中和ES6手册中所学的内容实现上述需求, 并使用 npm run test 命令通过测试
